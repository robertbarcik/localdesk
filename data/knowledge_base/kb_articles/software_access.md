# Software Access Requests

**Article ID:** KB-004
**Category:** Access
**Last Updated:** February 2026

## How to Request Software

### Standard Software (Pre-Approved)
The following software is pre-approved and can be installed from the Company Software Center without a ticket:
- Microsoft Office 365 suite
- Slack (desktop client)
- Zoom
- FortiClient VPN
- 7-Zip
- Adobe Acrobat Reader
- Firefox / Google Chrome

**To install:** Open the **Company Software Center** application on your machine → browse or search → click "Install".

### Non-Standard Software
For software not in the Software Center, you must submit a request:

1. Go to the IT self-service portal.
2. Select **"Software Request"** from the catalog.
3. Fill in:
   - Software name and version
   - Business justification (why you need it)
   - Whether it requires a paid license
4. Your manager will receive an approval request automatically.
5. After manager approval, IT Security reviews the software for compliance and security.
6. If approved, IT will either add it to your Software Center or install it remotely.

**Typical turnaround:**
- Free/open-source tools: 2-3 business days
- Licensed software: 5-7 business days (license procurement may take longer)
- Software requiring security review: up to 10 business days

### Software That Requires Special Approval
The following categories require CISO approval in addition to manager approval:
- Development tools with network access (IDEs with plugins, package managers)
- Remote access tools (TeamViewer, AnyDesk — VPN should be used instead)
- Encryption/security tools
- Any software that processes personal data (GDPR review required)

## License Management
- Do not share license keys between users
- If you no longer need licensed software, notify IT so the license can be reassigned
- Software audits are conducted quarterly — unlicensed software will be removed

## When to Contact IT
- If the Software Center is not working
- If installation fails with an error
- For urgent software needs that cannot wait for the standard process
